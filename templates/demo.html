Our Work

This website application provides a auto design tool for mass timber rocking wall system, this procedure will optimize desin for mass timber rocking wall systems utilizing a genetic algorith (GA) beyond typical displacement-based design metrics for wood buildings. Users only need to input a few parameters of the building and design targets they want to design, and the application will automatically generate optimized design result.
